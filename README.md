# akademi-cilik
Akademi Cilik adalah aplikasi web edukasi interaktif berbasis React dan TypeScript. Aplikasi ini menyediakan modul pembelajaran komprehensif dan kuis untuk 9 mata pelajaran: Energi &amp; Al-Qur'an, Matematika, IPA, IPS, Sejarah, Bahasa Indonesia, Bahasa Inggris, Tajwid, dan Komputer Dasar.
